# To-Do List com JavaScript

Aplicação simples de lista de tarefas desenvolvida com **HTML, CSS e JavaScript **, com persistência de dados no navegador.

---

## Funcionalidades

- Adicionar novas tarefas  
- Remover tarefas  
- Marcar tarefas como concluídas  
- Persistência com `localStorage`  

---

## Tecnologias

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage
